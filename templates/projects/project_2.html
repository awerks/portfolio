Overview

Campus Scheduler is a web-based platform designed to help businesses, particularly Small to Medium Enterprises (SMEs), Startups, and Tech companies, manage their workflows effectively through a role-based interface (Admin, Manager, Worker ). Hosted on Railway, it simplifies project management, event scheduling, task assignment, and team communication.

Users can register directly or sign in using Google Authentication. The system uses a PostgreSQL database for robust data storage and utilizes a tech stack chosen for rapid development and flexibility:

Frontend: HTML, CSS, JavaScript.
Backend: Python (Flask), using REST APIs.
Database: PostgreSQL.

The platform aims to provide a cost-effective solution, initially free, with potential for a future subscription model. It was made with learning in mind.
