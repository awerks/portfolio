Project:

Personal Photography Portfolio

Built With:

HTML, CSS, JavaScript, Tailwind CSS

Overview

This project is a web-based portfolio designed specifically to showcase personal photography work in a visually appealing and organized manner.

The project heavily uses Tailwind CSS for rapid, utility-first styling.

Key third-party tools like FancyBox are integrated for enhanced image viewing (lightbox galleries), jQuery supports some JavaScript interactions, and Google Fonts are used for refined typography.
